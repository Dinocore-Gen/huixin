# Music Folder

Place your background music file here.

**Required filename:** `background-music.mp3`

The website will automatically detect and play this file when the "Open the Surprise" button is clicked.

## Where to get free music:
- [Pixabay Music](https://pixabay.com/music/) — search "soft piano" or "cute instrumental"
- [Free Music Archive](https://freemusicarchive.org/)
- [Incompetech](https://incompetech.com/)

Recommended: A soft piano or music box instrumental (1-3 minutes, loop-friendly).
